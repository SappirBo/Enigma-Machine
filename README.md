## Combination
1. Rotors: 5 * 4 * 3 = 60.
2. Starting: 26^3 = 17,576.
3. Plugboard: 26! / (6! * 10! * 2^10) = 150,738,274,237,937,250.
4. Total: 158,962,555,217,826,360,000.

Means that there are more then **158 Quintillion** ways to set up the Enigma Machine!
